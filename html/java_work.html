JAVA开发工程师

其他职位

8k-12k /北京 / 经验1-3年 / 本科及以上 / 全职

职位描述：

  • 1.根据需求文档，熟练使用Axure等原型工具快速明确完成需求分析及原型开发；
  • 2.根据需求分析文档完成数据库及程序设计；
  • 3.制定开发计划并能够分析解决软件开发过程中的问题。

任职要求:

  • 1.计算机相关专业专科及以上学历，3年以上Java Web开发工作经验；
  • 2.熟练应用多种JAVA设计模式，并在实际的项目中有比较丰富的应用经验；
  • 3.熟悉Java编程语言，有J2EE相关技术开发经验，对struts2、mybatis、spring等开源技术框架有实际项目经验，熟练掌握基于Java的Web开发；熟悉Linux操作系统配置管理；熟悉javascript、h5等前端开发技术；
  • 4.掌握MySQL数据库的开发、配置、管理、调试，熟练掌握SQL查询优化及存储过程编写；
  • 5.熟练使用svn、git等版本管理工具；熟练使用Axure等原型工具；
  • 6.强烈的责任心和良好的分析问题、解决问题的能力，良好的沟通能力，良好的职业道德及敬业精神；
  • 7.有微信平台开发、微信应用开发、H5、微站等移动互联网项目开发经验者优先。
